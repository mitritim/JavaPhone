Project: JavaPhone
Authors: iGroup
Version: 2014-01-12

This program enables the users to get an image of a scanned contract
and then copy the customer data in a form.
The data is saved to a xml file and is shown in the program.
The program also enables to change user data and add users
as well as see all available services and change the price.

How to start the project:
call the main method in the class JavaPhone.
